# Atlas AI OS

Локальная операционная система для управления AI‑персонажами и производством контента.

## Запуск на Mac

```bash
git clone https://github.com/zakutailopavel-cyber/Atlas-AI-OS.git
cd Atlas-AI-OS
npm install
npm run dev
```

Откройте адрес, который покажет Terminal (обычно http://localhost:5173).

Данные MVP сохраняются локально в браузере. Файл `.env` не попадает в GitHub.
